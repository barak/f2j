Java numerical software links

Java has generated a substantial amount of interest from the numerical computing community. In addition to JLAPACK, several companies and research groups are actively working on Java numerical software.

    • Eigenvalue routines

  • Synopsis: http://www.vni.com/index_hi.html Visual Numerics JNL, a numerical library for Java.
