Java code

After the parser constructs an AST, the root of the tree is passed to code generation procedures to generate Java source code or Jasmin opcode. Since the FORTRAN source is assumed syntactically and semantically correct, the tree is recursively traversed without formal semantic checking. The Java source code generation is done in two passes. The first pass performs basic type assignment and the second pass emits the source code.
