    Bytecode Translation

  • f2java translates GOTO statements as calls to a dummy method, for example: Dummy.go_to(100)
  • f2java also translates labels to method calls: Dummy.label(100)
  • Compile Java source code as usual with javac.
  • Transform calls to the Dummy class into real goto instructions using the bytecode transformer.
