    COMMON Statements

  • Used in BLAS/LAPACK test routines.
  • If variable names in multiple COMMON block declarations differ, the variable names are merged into one name.

    public class dblat1_combla
    {
    static int icase = 0;
    static int n = 0;
    static int incx = 0;
    static int incy = 0;
    static int mode = 0;
    static boolean pass = false;
    }

    COMMON /COMBLA/ICASE,
    $ N, INCX, INCY, MODE, PASS
