    Programming Interfaces

  • Raw interface
  • Simplified interface

  • User directly calls the underlying numerical routine
  • User passes in column-major array
  • 2D arrays are mapped to 1D
  • User passes in separate offset parameter for each array

  • User calls a simplified version of the original routine
  • The simplified interface takes care of re-majoring the arrays and calling the underlying routine
  • User passes in Java-style 2D arrays in row-major layout
  • Offset parameters eliminated for arrays (assumed to be zero) to simplify calling sequence
