    DATA Statements

  • Translated to Java variable declaration with inline initialization.
  • Use static block of the Java class for cases that can’t be translated as mentioned above.

    INTEGER X(10)
    DATA X(3),X(7)/1,2/

    static int x[10];
    static {
    x[3-1] = 1;
    x[7-1] = 2;
    }

    INTEGER X
    DATA X/99/

    static int x = 99;
