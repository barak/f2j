    Variable Translation

  • All variables are translated to static in Java - don’t worry about SAVE statements.
  • Variables in the Fortran source corresponding to Java reserved words must be transformed (e.g. import Import).
  • All Fortran character types are translated to Java String objects.
  • PARAMETER ‘variables’ are generated as “public static final” in Java.
