    Programming Interface

  • BLAS and LAPACK routines are generated as single classes containing the procedure as a single static method.
  • Methods are invoked using the class name method name convention, e.g., Ddot.ddot.
  • Routines are linked in by importing from the org.netlib.blas and org.netlib.lapack packages.
