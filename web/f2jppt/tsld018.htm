    EQUIVALENCE Statements

  • f2java can handle equivalenced variables, provided that:

    INTEGER X,Y
    DOUBLE PRECISION A,B
    EQUIVALENCE (X,Y), (A,B)

  • The variables have the same data type
  • If the variables are arrays, the start index of each must be 1 (that is, you can’t equivalence the start of one array to the middle of another).

    static int x_y= 0;
    static double a_b= 0.0;

    INTEGER X(3,3),Y(9)
    EQUIVALENCE (X,Y)

    static int [] x_y= new int[(3) * (3)];
